# guia-estudante-unama
Projeto Vue.js - Guia do Estudante UNAMA

Este é um projeto desenvolvido em **Vue.js** que simula um portal acadêmico para estudantes da UNAMA.  
O sistema organiza informações sobre disciplinas, biblioteca, atividades complementares, provas e avaliações, além de uma seção sobre o curso.

---

## Funcionalidades

- **Home (Disciplinas):**  
  Lista de disciplinas com professor, horário e status.  
  Possui busca dinâmica para encontrar disciplinas rapidamente.

- **Biblioteca:**  
  Catálogo de livros de programação com imagens e informações relevantes.

- **Atividades Complementares:**  
  - Barra de progresso mostrando horas concluídas.  
  - Formulário para cadastrar novas atividades.  
  - Lista organizada com status (Validada, Em análise, Pendente).

- **Provas e Avaliações:**  
  - Separação por semestre (até o 3º).  
  - Provas realizadas com data e nota.  
  - Provas pendentes destacadas no 3º semestre.

- **Sobre o Curso:**  
  Informações detalhadas sobre o curso de Ciência da Computação, organizadas em tópicos.
